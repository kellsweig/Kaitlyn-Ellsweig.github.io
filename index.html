The Friends of the Cobbossee Watershed

Combining machine learning and data analysis to provide insights to nonprofit organization

I worked with The Friends of the Cobbossee Watershed, an environmental nonprofit working to protect the region's waterbodies from pollution and the threat of invasive species. I first used machine learning to develop a Gradient Boost model to predict the number of Courtesy Boat Inspections the organization would conduct in different scenarios. I then created multiple scenarios and deployed the model on those scenarios to create labor and budget projections. Using these projections and other analysis, I drafted recommendations for changes to budget and staffing for 2024.

Reddit Post Classification Project

Utilizing NLP and Machine Learning to Classify Posts

In this project I developed a classfication model that was able to determine with 99.6% accuracy which of two related subreddits a post came from. First, I used the Reddit API, PRAW, to collect reddit posts from the subreddits r/ultralight and r/camping. I used a TF-IDF vectorizer for NLP and then trained a Random Forest, Classifier, AdaBoost Classifier and Voting Classifier to achieve my final model.
